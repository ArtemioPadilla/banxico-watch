Run `npm run monday` and surface the dashboard it prints.

```bash
npm run monday
```

After the script runs, present its sections as a numbered menu (1/2/3) and
ask the user which item to act on. Do NOT automatically pick — the whole
point of the Monday ritual is the dev consciously chooses where to spend
the cycle.
