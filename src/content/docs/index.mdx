## What's inside

This scaffold is opinionated. It ships with the stack, the workflow, the
ethics framework, and the visual-regression harness already in place — so
your first commit is feature work, not config.

### Stack

Astro 5, React 19, Tailwind v4, shadcn/Base UI (not Radix), TanStack
Table + Query + Virtual, Tremor Raw KPIs, Recharts, Motion (post-Framer),
`@vite-pwa/astro`, Nano Stores. → [Stack overview](/docs/stack/overview/)

### How we work

Shape Up cadence (not Scrum), red→green TDD with `Tdd-Red:` commit
trailers, Zod-driven Spec-DD, Claude Code orchestration with three
sub-agents. → [Inceptor](/docs/how-we-work/inceptor/)

### Ethics & UX

8-item ethics checklist grounded in BJ Fogg's *Persuasive Technology*.
Tiered enforcement: tier-0 skips docs PRs, tier-1 covers UI tweaks,
tier-2 covers new persuasive surfaces. Mechanical `risk:high` triggers.
→ [Persuasive-tech framework](/docs/ethics-ux/framework/)

### Building

How to add a component, how hydration works, what the compound-component
gotcha is, how to theme, how to test. → [Adding a component](/docs/building/adding-a-component/)
