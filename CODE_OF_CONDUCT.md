# Code of conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Assume good faith.
- Critique ideas; don't belittle people.
- No harassment of any kind.
- Disagreements happen — argue the work, not the worker.

## Reporting

If you experience or witness unacceptable behavior, report it via the same
private channel listed in [`SECURITY.md`](./SECURITY.md). Reports are kept
confidential.

## Enforcement

Maintainers may warn, temporarily ban, or permanently ban contributors who
violate this code. Decisions are recorded as ADRs in `docs/decisions/` when
they set precedent.
